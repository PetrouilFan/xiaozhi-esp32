# v3.0.1 Plan: Robot Face Procedural Display

## Overview
Replace the current emoji/asset-based display system with a **procedural animated robot face** using real-time computed eye shapes. This eliminates dependency on SPIFFS/LittleFS asset storage for graphics and provides smooth, state-reactive animations.

## Goals
1. Zero external graphics assets (no emoji files, no GIFs)
2. Real-time procedural eye generation from scalar parameters
3. State-driven animations (listen, think, speak, sleep, emotions)
4. Transfer-efficient: all data is 14 floats per eye
5. 30 FPS on ESP32-S3 with minimal CPU load

---

## Architecture

```
main/display/procedural/
├── types.h              # Core structs (EyeParameters, FaceState, KeyFrame)
├── easing.h             # Interpolation functions (linear, quad, bounce)
├── timeline.h/.cc        # Keyframe track evaluation
├── scheduler.h/.cc       # Animation clip composition + priority
├── eye_shape.h/.cc       # Polygon contour generator (16-point capsule)
├── animation_library.h/.cc # Preset clips: blink, boot, happy, sad, etc.
├── procedural_display.h/.cc # Main display class (inherits Display)
└── eye_anim.cc           # Convenience helpers
```

## Data Flow

```
Application events (listen/speak/sleep)
        |
        v
+---------------+
|  Face State   |  <-- StateMachine (BOOTING->IDLE->LISTENING->SPEAKING)
+---------------+
        |
        v
+---------------+
|   Scheduler   |  <-- Priority clips: blink, emotion, transitions
+---------------+
        |
        v
+---------------+
|   Timeline    |  <-- Keyframe evaluation at 30 FPS
+---------------+
        |
        v
+---------------+
|  Eye Shape    |  <-- Generate 16-point polygon from parameters
+---------------+
        |
        v
+---------------+
|   LVGL Canvas |  <-- Draw filled polygon
+---------------+
```

## Eye Parameters (14 floats per eye)

| Parameter | Range | Effect |
|-----------|-------|--------|
| center_x, center_y | +/- 4 | Eye position offset |
| scale_x, scale_y | 0.5-1.5 | Eye size |
| rotation_deg | +/- 15 | Eye tilt |
| inner_corner_raise | -1 to 1 | Inner eyelid corner height |
| outer_corner_raise | -1 to 1 | Outer eyelid corner height |
| top_cut | 0-1 | Upper eyelid closure |
| bottom_cut | 0-1 | Lower eyelid closure |
| roundness | 0-1 | Corner sharpness |
| brightness | 0-1 | Eye glow intensity |
| visible | bool | Show/hide |

## Animation Clips (Preset Library)

| Clip | Duration | Description |
|------|----------|-------------|
| boot_open | 0.8s | Eyes open from closed |
| blink | 0.25s | Quick eyelid close/open |
| happy | 0.5s | Eyes squeeze to arc |
| sad | 0.6s | Outer corners droop |
| angry | 0.4s | Inner corners lower |
| surprised | 0.4s | Eyes widen |
| thinking | 0.5s | Subtle squint |
| sleep_enter | 1.0s | Eyes close and dim |
| wake_enter | 0.8s | Eyes open and brighten |

## State Machine

```
BOOTING --(timer)--> IDLE
IDLE --(wake_word)--> LISTENING
IDLE --(sleep_cmd)--> SLEEPING
LISTENING --(speech_stop)--> THINKING
THINKING --(playback_start)--> SPEAKING
SPEAKING --(playback_stop)--> IDLE
SLEEPING --(wake_word)--> WAKING
WAKING --(timer)--> IDLE
```
